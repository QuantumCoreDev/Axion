# How Axion Works

**Axion** is a Quantum Random Number Generator built using:

- A laser diode and a 50/50 beam splitter to split photons
- Two BPW-Z photodiodes to detect which path a photon takes
- ADS1115 16-bit ADC to read analog voltages
- ESP32 microcontroller running MicroPython
- Von Neumann method to remove bias
- OLED display to show 64-bit quantum entropy
- Wi-Fi Dashboard to stream randomness to browser

Axion uses the true unpredictability of quantum physics to generate real, pure randomness — not just software-based pseudo-random numbers.
